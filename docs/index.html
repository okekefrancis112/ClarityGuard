Custom Parser

Hand-written S-expression tokenizer and recursive descent parser. Produces a fully typed AST with precise line and column positions for every node.

Pluggable Detectors

Each detector implements a simple interface: receive the AST, return findings. Add new detectors without touching existing code.

Zero Dependencies

No runtime dependencies. Pure TypeScript compiled to JavaScript. The entire scanner is 14KB bundled and minified.
